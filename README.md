# shapeyourcity-rezoning
Tool to web scrape active rezoning applications in Vancouver, BC.<br>
Output is given either in the console or in a text file.<br>
The output is a list of each application's address and a link to the application.<br>
Data is fetched from the [Shape Your City](https://www.shapeyourcity.ca/rezoning) website.<br><br>
Uses python and the requests library, which must be imported with ```pip install requests```
